== これなに？ ==

ロボカップの3Dサッカーシミュレーションのエージェント開発用のツールです。

このツールを使用することで、Naoモデルのポーズとモーションを作成することができます。

== 内容物 ==

この頒布物には以下のものが含まれています。

 * ReadMe.txt
   * このファイルです

 * same.jar
   * モーションエディターの本体です

 * NOTICE
   * ApacheLicenseによって求められているNOTICEファイルです

 * ApacheLicense.txt
   * ApacheLicenseによって求められている、使用ライブラリのライセンスコピーです

 * LICENSE
   * この頒布物のライセンスを示すファイルです。

この頒布物はApacheLicense 2.0によって頒布されているライブラリが含まれていますが、この頒布物自体のライセンスは２条項BSDライセンスです。


== 必要環境 ==

 * Java Runtime Environment 6以上
 * simspark 0.6

== 起動方法 ==

 * simspark 0.6を起動する
 * same.jarのあるディレクトリで java -jar same.jar とコマンドを打つ

== 使い方 ==

 * 起動時にポーズとモーションを保存するディレクトリを聞かれるので選択してください。
